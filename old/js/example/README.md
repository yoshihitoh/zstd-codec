# An Example of zstd-codec on browser

## Installation
```bash
yarn install
```

## Build
```bash
yarn build
```

## Run

Run HTTP server
```bash
# move to dist
cd dist/

# with python2.x
python -m SimpleHTTPServer 8084

# with python3.x
python -m http.server 8084
```

Access to `http://localhost:8084`
